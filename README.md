# PassEngine 🔐

PassEngine is a robust, terminal-based password generator built with Python. It generates strong, hack-proof passwords of custom lengths and maintains a complete execution history.

## Features
- **Custom Length Control:** Specify any password length from 4 to 1000 characters.
- **History Tracking:** Automatically saves generated passwords to memory and an external text file (`passEngineHistory.txt`).
- **History Management:** View or clear past generated passwords directly from the interactive menu.
- **Styled Terminal UI:** Clean, color-coded text interface using ANSI escape sequences.

## How to Run
`python PassEngine.py`
